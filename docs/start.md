## Quick start

01.get the code

```html
git clone https://github.com/YMC-GitHub/lib-doc-tpl-with-vuepress.git
```

02.install his dep

```html
npm install
```

03.run in dev mode

```html
npm run docs:dev
```

04.run in pro mode

```html
npm run docs:build
```

05.release to you platform

```html
./relaseh.sh
```
